Website Development

  1. Website apa yang paling ingin kamu buat? Tentang apa?

    2. Saya ingin membuat website e-commerce dan social media yang akan digabungkan dalam 1 website

  2. Hasil atau informasi apa yang paling ingin kamu sampaikan?

    3. Website tersebut seperti pada e-commerce lainnya yang digunakan untuk transaksi jual beli seperti tokopedia, bukalapak dan lainnya tetapi di dalam website tersebut ada social media seperti twitter, facebook, path untuk user dapat memposting di social media tersebut. Jadi ada interaksi antara penjual dan pembeli serta dapat memposting foto tersebut.

  3. Bagaimana kira-kira kamu memproses pembuatan website dari awal hingga akhir?

    1. Pengumpulan Informasi / Data

      2. Tahap pertama, dalam membuat situs web adalah mengumpulkan informasi sebanyak mungkin. Dalam bisnis, e-commerce ini apa yang dibutuhkan oleh konsumen dan apa yang paling dicari oleh konsumen. Menentukan tujuan, sasaran, target visitor dan konten dari website e-commerce tersebut.

    2. Planning / Perencanaan

      3. Tahap kedua, Setelah mengumpulkan informasi, lalu membuat sitemap atau peta situs, memilih teknologi apa yang akan digunakan dan membuat mockup design website untuk memvisualisasikan rancangan design tersebut

    3. Development / Pengembangan

      4. Tahap ketiga, Setelah rancangan design sudah disetujui atau disepakati, maka pengembangan website dimulai. Semua informasi dan design yang telah dilakukan lalu akan dikembangkan dengan bahasa pemrograman agar design tersebut dapat direalisasikan dalam website online.

    4. Testing / Pengujian

      5. Tahap keempat, Setelah pengembangan website selesai dilakukan maka pengujian website harus dilakukan untuk melihat bawha fitur-fitur dalam website tersebut telah sesuai dan bekerja sebagaimana yang telah direncanakan. Apabila seluruhnya telah selesai maka website tersebut siap untuk di launching online.

    5. Maintenance / Pemeliharaan

      6. Tahap kelima, maintenance / pemeliharaan sebuah website perlu dilakukan untuk memastikan pada website berjalan online tersebut user tetap aman dan berjalan secara optimal.
