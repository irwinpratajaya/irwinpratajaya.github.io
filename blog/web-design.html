Website Design

  1. Apa itu adaptive dan responsive web design? Apakah kamu setuju bahwa teknik tersebut bagus? Mengapa atau mengapa tidak?

    2. Adaptive dan responsive design adalah sebuah metode yang memungkinkan tampilan website yang dibuka di berbagai device (Laptop, Smartphone dan Tablet) mengikuti ukuran dari device tersebut. Interface atau tampilan website tersebut lebih optimal dan akan menyesuaikan dengan device yang digunakan dan membuat user lebih mudah dan nyaman.
    Ya, saya setuju bahwa adaptive dan responsive design adalah teknik yang bagus. Tampilan responsive dan adaptive design membuat user lebih nyaman untuk menggunakan device apapun untuk mengakses sebuah website karena ukuran website dapat berubah sesuai dengan device yang digunakan.

  2. Apakah hadirnya smartphone dan tablet mengubah desain web secara signifikan? Bagaimana kira-kira teknologi baru akan mengubah desain web di masa depan?

    3. Tentunya dengan Hadirnya smartphone dan tablet akan mengubah design website secara signifikan karena semakin banyak smartphone dan tablet yang ada dipasaran membuat design website mengikuti perubahan jaman tersebut, perubahan tersebut terlihat pada saat dulu beberapa website menggunakan sub domain untuk versi mobile (m.website.com) dan pada saat ini hal tersebut tidak perlu dilakukan karena adanya adaptive dan responsive design.
    Tekologi akan terus berkembang dan tidak akan pernah berhenti, seiring dengan semakin canggih teknologi tersebut maka web design akan terus mengikuti perubahan tersebut.

  3. Bagaimana kira-kira orang yang memiliki kekurangan fisik menggunakan website? Apakah kita perlu menyesuaikan website untuk mereka juga? Solusi apa yang dapat mengatasi keadaan tersebut?

    4. Untuk orang yang memiliki kekurangan fisik, saat ini sudah ada teknologi text to voice dan voice recognition (siri, google voice) pada smartphone yang ditujukan kepada orang yang memiliki kekurangan fisik.
    Sebuah website juga harus mempertimbangkan dan melihat siapa user yang akan mengakses website tesebut, kembali lagi kepada tujuan dibuatnya sebuah website, apakah kaum disabilitas akan mengakses website tersebut atau tidak, jika target user website salah satunya adalah kaum disabilitas, maka sebaiknya fitur-fitur untuk mereka juga disediakan agar mereka nyaman saat mengakses website tersebut.

  4. Ide atau konsep apa yang sesuai denganmu saat mempelajari desain web?

    5. Saya ingin membuat website yang mudah untuk digunakan dengan tampilan yang menarik sehingga visitor tidak bosan pada saat mengakses website saya.
